The Big Web Show is a podcast about websites! It’s hosted by Jeffrey Zeldman, a web designer and author who also founded the web-based magazine A List Apart, which I’ve referred to in the first week of this course. I recommend this episode, where Jeffrey interviews Jen Simmons, a designer who coined the term “Intrinsic Web Design,” which many are calling the next evolution of the web after responsive web design.

Intrinsic web design: Jeffrey Zeldman interviews Jen Simmons , The Big Web Show podcast, April 27 2018.
