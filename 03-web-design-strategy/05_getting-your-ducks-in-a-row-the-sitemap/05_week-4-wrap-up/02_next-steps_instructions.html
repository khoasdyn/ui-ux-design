If you’ve finished this course, I encourage you to build on your UI design skills by enrolling in the next one offered in the UI/UX Specialization , Web Design: Wireframes to Prototypes . This course is focused on the application of the early UX research to actual user interfaces: the creation of wireframes, high-fidelity mockups, and clickable prototypes. Along the way we will also discuss: responsive web design and mobile web challenges; mobile-first approaches to web design; web typography; the relationship between design and programming and whether it is important to know how to code; and finally the different web technologies that make the web work, such as HTML, CSS, JavaScript, server-side coding, and databases.

I hope to see you in the next course.
