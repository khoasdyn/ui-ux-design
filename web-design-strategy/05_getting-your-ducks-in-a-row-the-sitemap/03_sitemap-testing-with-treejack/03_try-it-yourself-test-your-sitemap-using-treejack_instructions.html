Using Treejack to test user paths is very straightforward, and I encourage you to try it out on the sitemap you're developing in this course.

To set up a test in Treejack, first you’ll need to input your “tree”—or rather, your sitemap—into the tool.

Then, define what tasks you want to test with your users. Some possible questions you might ask are:

  • How can I place an order for food?

  • Where can I find the restaurant’s hours of operation?

  • How can I find out if the restaurant delivers to my area?

Finally, recruit participants for your test by sending friends and family a link to your survey.

For more thorough explanation of how to set up a test in Treejack and read the results, visit: https://www.optimalworkshop.com/101/tree-testing .
