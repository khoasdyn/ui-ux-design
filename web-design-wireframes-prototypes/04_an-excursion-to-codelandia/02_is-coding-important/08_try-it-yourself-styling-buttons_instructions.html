In this exercise, you will design 3 different buttons by writing CSS code. To begin, create a new folder on your computer and download the following three html/css files into this folder. Some browsers, such as Chrome, might rename the downloaded files, so make sure that the names of the files downloaded to your device are exactly the same as the ones below:

Navigate to your new folder and open the 'buttons.html' file. It should automatically open in your default web browser. Note that I already styled the first button. After you’ve completed this exercise, each of your buttons should have its own distinct visual style and a distinct hover effect.

The only file that you need to change is ‘your-styles.css.’ Navigate back to your new folder and open ‘your-styles.css’ in your operating system’s text editing program (Mac: TextEdit, Windows: Notepad). You can also use a code editor (Atom, Brackets, Sublime Text, VS Code).

Once ‘your-styles.css’ is open in your text editor, try altering any of the following CSS properties:

  • color

  • background-color

  • font-size

  • font-weight

  • text-transform

  • letter-spacing

  • border-radius

  • box-shadow

  • text-shadow

  • padding

  • border

  • transition

To view the changes, save the ‘your-styles.css’ file and refresh the ‘buttons.html’ file in your web browser. Remember that you can also use the web inspector of your browser to try stuff out on the fly. Just copy any changed properties back to your CSS file.

If you don’t know what the above mentioned properties do, research them on the internet. Here are two reference sites that should be helpful:
