There are plenty of elements and instances of interactivity or feedback right here. Without navigating away from this page, see if you can identify:

  • a call to action

  • icons as labels for different kinds of content

  • icons (symbols) used as buttons to register a user’s opinion, or report a problem

  • state changes or cues when you hover over an element

  • state changes to show whether an item has been completed or not

  • navigational elements or menus that appear when you need to use them

  • changes to the overall UI in order to focus on one particular element
