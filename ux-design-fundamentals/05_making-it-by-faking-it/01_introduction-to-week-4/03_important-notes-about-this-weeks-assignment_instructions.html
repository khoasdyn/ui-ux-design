At the end of this week you are asked to pull together your work to create a clickable prototype.

As you'll see in the lectures, I'm building my prototype using screens with a fully developed look and feel. You are not required to do this —you can build a prototype from wireframes as long as the user can complete the entire pathway as you've outlined it.

Second, you can use any software to create your prototype as long as you can generate a sharable link for peer review. In this course, we're using Figma; Marvel is a great alternative. Please be sure to test your prototype in your browser first before uploading the link to the platform so you can be sure it functions properly .
